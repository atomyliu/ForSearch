protected void btnSearch_Click(object sender, EventArgs e)
        {
            MemberServClient msc = new MemberServClient();
            string type = ddlist.SelectedValue.ToString();
            string kw = tb1.Text.ToString();
            int start = 0;
            int size = 10;
            int count = msc.GetCount(kw, type);
            lbCount.Text = count.ToString();
            Model.Member[]mem = msc.GetList(kw,type,start,size);
            RpMem.DataSource = mem;
            RpMem.DataBind();
            msc.Close();
        }

方法说明

?
1
GetList(string keyword, string type, int start, int size)
根据条件返回结果列表。keywork:关键字；start:分页开始位置；size:条数；type:查询的类型，也就是字段，默认为空，如果不为空需要传入固定值：（Enterprise,Products,Memo,Address,AdminName,WebName）
?
1
GetCount(string keyword, string type)

根据条件返回结果总数。keywork:关键字；type:查询的类型，也就是字段，默认为空，如果不为空需要传入固定值：（Enterprise,Products,Memo,Address,AdminName,WebName）

MemberContactServ 服务方法同上，其中type：ID,MemberID,Linkman,Phone,Fax,Mobile,Email

注：在ES中分页使用方法是从第几条开始查询几个，比如start=0，size=10，这样就是从第0条开始查询10条数据，所以如果需要继续查询，那么start=10，size=10，这样就是从第10条开始查询再查10条。

当然，也可以先通过GetCount()方法查询总数后在前台进行分页，但考虑到服务器压力和性能问题，建议不要这样做。
